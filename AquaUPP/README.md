# AquaUPP

Repositorio del frontend del proyecto **AquaUPP**, una Progressive Web App orientada al monitoreo y gestión de captación de agua de lluvia en comunidades rurales.

## Descripción

Este repositorio corresponde a una **reconstrucción funcional del prototipo**, basada en las interfaces definidas y en la documentación del proyecto (SRS, SCRUM y Arquitectura).

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Login con PIN (simulado)
- Dashboard informativo
- Registro manual de captación de agua
- Visualización básica de estadísticas
